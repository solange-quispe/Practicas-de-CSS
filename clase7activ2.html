Que es un REPOSITORIO?? Una carpeta que contendra el codigo fuente de nuestro desarrollo de software (sitio web, desarrollo mobile, etc.) Que es una VERSION?? Una version es un conjunto de archivo (codigo fuente), sobre los cuales se ha decidido no realizar mas modificaciones. El repositorio principal es el REPOSITORIO REMOTO Para poder subir repositorios necesito: Registrarme en GitHub Crear un repositorio nuevo haciendo click en 'now' que nos va a dar una ruta, nos copiamos esa ruta Luego descargo Git Bash, que es una consola de comandos Ingreso a la carpeta donde quiero guardar mi repositorio y abro mi git bash desde ahi Desde ahi escribo git clone y pego la ruta que copie previamente. Despues tengo que ingresar a esa carpeta que cree y para eso debo poner cd mas el nombre de mi carpeta que acabamos de crear Luego hacemos git add Luego hago git status y me tiene que salir en verde lo que queria agregar git commit = trakear $ git commit -m "este es mi primer cambio" verifico siempre con $ git status Y para terminar de subir mi repositorio = archivo debo usar $ git push -u origin master Vuelvo a corroborar con "$ git status" e ingreso en mi nuevo repoisitorio en este caso miambulacia y ahi deberia estar la carpeta que subi Para poner mi nombre hago lo siguiente: $ git config --global user.name "Austry" Para agregar mi correo electronico hago: $ git config --global user.mail "castilloaustry@gmail.com" Para ver lo que se hizo escribo $ git log, ahi me va a salir todos los commit que se hicieron, quien lo hizo, la fecha, la hora
